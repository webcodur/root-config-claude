# Skills (자동 활성화 스킬)

## 개요

`skills/` 디렉토리에 정의된 스킬은 특정 조건이 감지되면 자동으로 활성화된다. 명시적으로 호출하지 않아도 관련 작업 시 해당 지시문이 적용된다.

---

## 현재 파일 목록

```
skills/
└── nextjs-page/
    └── SKILL.md    # Next.js 페이지 생성 시 pages/views 분리 구조 적용
```

---

## 디렉토리 구조

각 스킬은 별도 폴더로 구성하며, 반드시 `SKILL.md` 파일을 포함해야 한다.

```
skills/
└── skill-name/
    └── SKILL.md    # 스킬 정의 파일 (필수)
```

---

## 파일 형식

```markdown
---
name: 스킬 이름
description: 스킬 설명 및 활성화 조건
allowed-tools: 사용 가능한 도구
---

# 스킬 지시문

(적용할 규칙, 패턴, 코드 예시 등)
```

### 프론트매터 필드

| 필드 | 필수 | 설명 |
|------|------|------|
| `name` | O | 스킬 이름 |
| `description` | O | 설명 및 활성화 조건 |
| `allowed-tools` | X | 사용 가능한 도구 제한 |

---

## 새 스킬 추가 방법

1. `skills/` 디렉토리에 스킬명 폴더 생성
2. 폴더 내에 `SKILL.md` 파일 생성
3. 프론트매터에 `name`, `description` 정의
4. 본문에 적용할 규칙, 패턴, 코드 예시 상세 기술
5. Claude Code 재시작 후 조건 충족 시 자동 활성화

---

## 스킬 vs 에이전트 vs 명령어

| 구분 | 활성화 방식 | 용도 |
|------|------------|------|
| 스킬 | 자동 (조건 감지) | 특정 패턴/규칙 자동 적용 |
| 에이전트 | Task 도구로 명시적 호출 | 전문 역할 위임 |
| 명령어 | `/명령어`로 명시적 호출 | 반복 작업 빠른 실행 |
