# file-history/

## 개요

파일 변경 이력을 저장하는 디렉토리이다. `Edit`, `Write` 도구 사용 시 변경 전 상태를 저장한다.

## 위치

```
~/.claude/file-history/
```

## 내용

- 파일 수정 기록
- 변경 전 스냅샷
- 롤백 정보

## 용도

- 파일 변경 실수 시 복구
- 변경 이력 추적
- 이전 버전과 비교

## 주의

- 파일 수정이 많을수록 용량 증가
- `.gitignore`에 포함되어 Git에 추적되지 않는다

## 삭제

```bash
rm -rf ~/.claude/file-history/
```

삭제해도 Claude Code 동작에 영향 없다. 단, 파일 변경 롤백 기능을 사용할 수 없게 된다.
