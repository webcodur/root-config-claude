# ide/

## 개요

IDE 연동 데이터를 저장하는 디렉토리이다. VS Code 등 IDE 확장 프로그램과의 통신에 사용된다.

## 위치

```
~/.claude/ide/
```

## 내용

- IDE 확장 연동 상태
- 에디터 상태 동기화
- 락 파일 (`*.lock`)

## 용도

- VS Code Claude 확장과 CLI 간 통신
- 에디터에서 선택한 코드 컨텍스트 전달
- 동시 실행 방지를 위한 락 관리

## 주의

- `*.lock` 파일은 `.gitignore`에서 제외된다
- IDE 확장 미사용 시 비어있을 수 있다

## 삭제

```bash
rm -rf ~/.claude/ide/
```

삭제 후 IDE 확장 재연결이 필요할 수 있다.
